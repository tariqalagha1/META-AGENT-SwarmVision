## 🎨 All About the Dashboard

### What Users See

**Left Panel**: System Graph
- Live topology of agents
- Color-coded health (cyan=good, amber=warning, red=critical)
- Real-time updates as status changes

**Top-Right Panel**: Alerts Stream
- Real-time anomalies auto-detected
- 5 heuristic categories (bottleneck, failures, patterns, correlation, load)
- Most recent alerts at top

**Middle-Right Panel**: Execution Timeline
- Step-by-step trace of what happened
- Event types: START, HANDOFF, DECISION, ANOMALY, ERROR
- Full JSON context available on click

**Bottom-Right Panel**: Decision Log
- Audit trail showing every AI decision
- Confidence scores visible
- Searchable and filterable
