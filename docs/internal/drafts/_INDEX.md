# Internal drafts

These documents predate the 2026-04-24 project walkthrough and describe
SwarmVision in commercial-product terms that do not match the current
developer-preview reality.

They are kept in version control because:
- The framing may become accurate as the product matures
- The text may serve as reference when writing future commercial pitches
- Deletion would lose the work that went into them

They are NOT authoritative descriptions of SwarmVision today. For current
state, see the repo root `README.md` and `docs/SETUP.md`.

Each file in this directory has been annotated with a status header
explaining when it was drafted and what its current status is.
